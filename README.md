# 負負得正

給國中小學生（六升七／國一）練習**正負數運算**的網頁小遊戲。對應台灣 108 課綱。

## 玩法

- 正負整數**加減法選擇題**，4 選 1
- **滾動式自適應難度**：連續答對自動變難、連續答錯自動變簡單
- **計分、星星、連勝**，連勝到 5 的倍數有星星獎勵
- 答錯時提供**解題講解**（特別加強「減去負數」的觀念）
- **音效**（可靜音）與**深色模式**
- 進度自動存在瀏覽器（單機）

## 技術

純前端（HTML / CSS / JavaScript），無需後端，可直接用 GitHub Pages 託管。

```
posneg-app/
├─ index.html
├─ css/style.css
└─ js/
   ├─ storage.js   存進度與設定
   ├─ sound.js     音效合成
   ├─ questions.js 出題引擎 + 自適應難度
   └─ main.js      流程控制 + 畫面
```

## 開發路線

- **Phase 1（完成）**：加減選擇題、自適應難度、計分星星連勝、答錯講解、音效、深色模式
- Phase 2：抽卡＋圖鑑、乘除法、計時模式、闖關
- Phase 3：帳號（帳號+PIN）與跨裝置同步（Firebase）
- Phase 4：老師／家長查看進度後台
- Phase 5：PWA 離線化、Capacitor 打包上架 Google Play
